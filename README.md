# Time Series Prediction on Crude Oil Rates using GRU

This project forecasts crude oil prices using Gated Recurrent Units (GRU), comparing a scratch-style GRU workflow and a TensorFlow/Keras GRU model, evaluated using RMSE.

## Tech
Python, TensorFlow/Keras, yfinance, NumPy, Pandas, scikit-learn, Matplotlib

## What’s inside
- `gru_big_data.ipynb` – notebook for data fetch, preprocessing, GRU training, evaluation, and plots
- `images/` – figures (RMSE curves, predicted vs actual)
- `requirements.txt` – dependencies

## How to run
```bash
pip install -r requirements.txt
